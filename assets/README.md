# Assets Directory

This directory contains app assets:

- **images/** - App icons, logos, and UI images
- **data/** - Sample data files for testing and development

## Images
Place your app images here:
- App logo
- Splash screen
- UI icons
- Educational diagrams

## Data
Sample data files for development and testing:
- `sample_questions.json` - Example questions in all four assessment types
- Import your own question banks here

## Note for Content Authors
Use the schemas defined in `IMPORT_SCHEMAS.md` to create your question banks.
